# Dotfiles

This dotfile is forked from Thoughtbot.

## Requirements

### zsh

```
chsh -s $(which zsh)
```

### dotfile

```
git clone git://github.com/tolearn2/dotfiles.git ~/dotfiles
```

### rcm

```
brew tap thoughtbot/formulae
brew install rcm
```

## Install

```
env RCRC=$HOME/dotfiles/rcrc rcup
```
